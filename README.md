Project Name is Gititools-platform

Description :
"I built a digital product marketplace using React where users can browse products, add them to cart, remove items, and see total pricing. I used React state management to handle cart logic, Tailwind CSS for UI, and deployed the project using Netlify."

Technology :
1/ React
2/ Tailwind
3/ JavaScript (ES6)
4/ DaisyUI
5/ Deploy Netlify

Features :
1/ Browse a collection of premium digital products with detailed information.
2/ Add products to the cart with a single click.
3/ View selected items in a dedicated cart section.
4/ Remove individual items from the cart.
5/ Clear all items from the cart using the checkout button.
6/ Automatically calculate and display total cart price.
7/ Toggle between product list and cart view seamlessly.
8/ Display real-time cart item count in the navigation bar.
9/ Show an empty cart message when no items are selected.
10/ Fully responsive design for mobile, tablet, and desktop devices.
